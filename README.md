# Implementasi-Interpolasi-Lagrange-Orde-Tinggi
Mini project metode numerik menggunakan Interpolasi Lagrange orde tinggi untuk melakukan pendekatan nilai fungsi pada titik f(0,5).  Perhitungan dilakukan dengan membentuk polinom Lagrange dari beberapa titik data hingga diperoleh estimasi nilai fungsi secara numerik menggunakan Microsoft Excel.
